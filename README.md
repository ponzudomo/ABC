## 概要
ABCの学習記録リポジトリです

自分用ではありますが、復習用にいくらかメモを残しています
せっかく興味を持って見に来てくださったなら、`notes/` `snippets/` あたりを読んでみるのが面白いかもしれません
ただ自分用に作ったものなのでまともに整備をしていません　悪しからず......

## リポジトリ構造

```markdown
.
├── `.codetest` ローカルにおいておけるコードテストもどき
├── `A`
|    ⋮   問題番号ごとに提出コードをまとめたフォルダたち
├── `E`
├── `notes/` 勉強メモ
│   ├── `mistakes/` やらかし集
│   │   ├── `A`
│   │   |   ⋮   問題ごとの反省メモ
│   │   ├── `E`
│   │   └── `checklist.md` 提出前の凡ミスチェックリスト
│   └── `tips` 汎用的なtips (より具体的なものは`snippets`にコードとしてまとめていく予定)
└── `snippets/` コード雛形一覧
    ├── `template.cpp` C++のテンプレ
    └── `template.kt` Kotlinのテンプレ
```
